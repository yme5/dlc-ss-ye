Encryption and Public Keys

Sehrish Saeed

Feb 28, 2021

Encryption

The internet is an open and public system that also exchanges private information. Encryption is what scrambles and changes a message to hide its true characteristics. Decryption unscrambles the message to make it readable. Keys allow users to unlock the “secret” message. The Caesar Cipher is an algorithm that is used to scramble messages. It uses numbers and letters to create encrypted keys. To make this algorithm harder to crack, one can add more numbers to create a large number of possibilities, but computers can crack these very quickly. 256-bit keys are used to make it harder for hackers to crack the message. Symmetric encryption is when the same key is used to encrypt and decrypt a message.

Public Keys

Public keys are used to encrypt data by anyone, but a private key can only be decrypted by a computer that has access. Public key cryptography is the foundation of all secure messaging on the internet. Secure Sockets Layer and Transport Layer Security and types of security protocols that protect users when browsing the web. The “lock” or https means that the computer is using public key encryption to transfer data.
