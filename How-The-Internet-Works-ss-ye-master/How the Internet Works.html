How does the Internet Work?

Yousuf Elkhoga

Mar 1, 2021

What Powers the Internet?

The internet works via an international collaboration of interconnected networks. As long as all these networks adhere to the open protocols defined by the Internet standards, host to host communication can be achieved with high amounts of reliability and in a minimal amount of time. Connection is achieved via direct means (DS, copper cable, fiber optic cable) or indirect means (WiFi, mobile networks). Access to the internet can be achieved via a web client, a type of software designed to access the internet. The client is connected to a local access network (LAN) via a router, and the LAN is connected to a wide area network via an Internet Exchange Point owned by an Internet Service Provider. Internet messages are sent along this network via packets, which are labeled pieces of information, to their destination address.

Why Use this Tech?

The most important part of Internet infrastructure is mutual buy in from all parties. In order to participate, a network must agree to use the protocols everyone else is using, or else communication breaks apart.

The Internet in Practice.

A coffee shop sets up an Access Point. Users can connect to this AP, whch is connected to a router in the back of the shop. The router is in turn connected to a IXP contorlled by a ISP that controls the traffic from inside the shop.
