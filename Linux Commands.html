Linux Commands

  • Pdw: allows the user to know which directory they are in
  • 1s: allows the user to know what files in the current directory
  • Cd: allows the user to go to a directory Ex: if the user is in the home folder and they want to go to the Downloads folder they can type “cd Downloads”
  • Mkdir & rmdir: mkdir is used when a user needs to create a new folder or a directory. Rmdir is used to delete a directory only if it is empty.
  • Rm: used to delete a directory or files
  • Touch: used to create a file
  • Cp: used to copy files through the command line
  • Mv: used to move files through the command line
  • Locate: used to locate a file in the Linux system similar to when using the search command in Windows.
