Tutorial: Connecting Docker in Pycharm

YousuF Elkhoga

Mar 22, 2021

  • Before beginning this tutorial, make sure Pycharm Pro, Docker, and Git are all installed.
    1. This tutorial will connect a github repository to Pycharm, although Git is not necessary
  • Clone the intended repository from Github into Pycharm by using the VCS feature
    1. For this Tutorial, all of the files (besides .gitignore and README) will be created in the tutorial
  • Begin by creating a source directory in the Pycharm Project. In this tutorial, it will be called “src”
  • In the src folder, create your python project
    1. In the case of this tutorial, it will be a simple file called my_script.py, and will contain this code
# Sample taken from pyStrich GitHub repository
# https://github.com/mmulqueen/pyStrich
from pystrich.datamatrix import DataMatrixEncoder
 
encoder = DataMatrixEncoder('This is a DataMatrix.')
encoder.save('./datamatrix_test.png')
print(encoder.get_ascii())
 
  • In the main directory of this project, create the Dockerfile
    1. New -> File -> Dockerfile
    2. Use the below code
FROM python:3
 
ADD src /src
 
RUN pip install pystrich
 
CMD [ "python", "./src/my_script.py" ]
  • Docker is now ready for setup
    1. Go to File -> Settings -> Build, Execution, Deployment -> Docker and Connect to the Docker Daemon
      • If using Docker Desktop, use the TCP socket with the Engine API URL tcp://localhost:2375. Make sure that it is enabled in Docker Desktop!
  • Go to the top right and click “Add Configuration”
    1. Add a run configuration by clicking the plus button
    2. Go down to where it says docker, and click dockerfile
    3. Select the Dockerfile created earlier in the dropdown
    4. Name the run configuration after the purpose of the app, in this case, barcode
    5. Click “Ok” to complete
  • Press the play button on the top right to begin running the image that was just created in the previous step
    1. On the bottom of the page,  the services window will appear, showing all the containers that are running, the build log, log, and the attached console.
  • On the bottom right of the page, click on “Python Interpreter” -> “Add Interpreter”
    1. Select Docker from the window, and make sure the right image is selected, click ok.
  • Click on interpreter again, and select the interpreter setting to make sure all the packages are there
  • Now Pycharm and Docker are connected, congrats!
