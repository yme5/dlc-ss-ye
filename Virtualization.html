What is Virtualization?

In IT, virtualization is the act of creating virtual environments or virtual versions of something which could include things like storage devices, hardware platforms, and computer network resources. Virtualization took off when it was the solution to two problems. Companies could now split their services and run legacy apps on many operating system types and servers started being used more efficiently which reduced the costs that came with maintenance. Virtualization allows the ability to create IT services that use resources that would normally be tied to physical hardware. A user can use a computer's full physical capacities by distributing the capabilities among different users and environments. In the past, virtual machines used to be bulky in size with unstable performance and long boot time processes. They also had problems in software updates and continuous delivery.
